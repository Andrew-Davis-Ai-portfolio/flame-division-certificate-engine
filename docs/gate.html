Why this gate exists

Certificate tools sit at a different level than public marketing pages. They represent completed training, verified work, and Flame Division signatures. The gate:

  • Prevents casual traffic from accessing internal artifacts.
  • Keeps certificate templates from being taken out of context.
  • Signals that certification is earned, not downloaded.

How access works

Once your code is accepted, this browser session stores a short-lived token in sessionStorage. Any gated page in this repo checks for that token before loading. Closing the browser or clearing site data will reset access and send you back to this gate.
